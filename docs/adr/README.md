# Architecture Decision Records

Required when introducing a **new pattern** that diverges from [agent-os/CODE_STANDARDS.md](../../agent-os/CODE_STANDARDS.md).

## Format

`NNN-short-title.md` (e.g. `001-deploy-strategy-registry.md`)

```markdown
# NNN. Title

Date: YYYY-MM-DD
Status: proposed | accepted | superseded

## Context

## Decision

## Consequences
```

## Rules

- Master + SM-α must acknowledge before merge
- Link ADR in PR Gate 2
