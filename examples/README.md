# DocsMint Examples

Focused examples live here. Keep each example small and centered on one product surface.

## Examples

- `minimal` — the smallest runnable project and init-parity smoke fixture.
- `i18n-versioned` — default-locale root docs, French fallback routing, and archived version docs.
- `custom-collections` — a custom `playbooks` collection beside docs and writing.

`apps/site` is the full dogfood documentation site. Use these examples when you want a narrow fixture.
