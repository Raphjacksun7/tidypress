# Custom Collections Example

This example adds a `playbooks` collection beside docs and writing.

It proves custom collection routing, index pages, and content entries under a custom base path.

Run from the repo root after `pnpm install`.

## Build

```sh
pnpm --filter @tidypress/example-custom-collections build
```

Output:

```txt
examples/custom-collections/docs/build/
```

Inspect:

```txt
/playbooks
```

Docs: <https://tidypress.pages.dev/docs/configuration>
