> Minimal markdown docs and writing.

DocsMint is for projects that need a public site without becoming a publishing operation.

Thesis:

- markdown as source
- docs and dated writing in one tree
- a clean default interface
- search at build time
- static output
- deploy anywhere

The tool should feel closer to a folder than a platform.

Write the pages. Build the site. Keep the shape simple enough to leave.
